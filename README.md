# Fast Reload


Простой аддон для PulseSync, который добавляет перезагрузку страницы по клавише F5 или по нажатию кнопки перезагрузки

поддержать разработчика -------->

https://clck.ru/3SzeqP (Boosty) =)

## Возможности

- Нажми **F5** для мгновенной перезагрузки страницы или нажми на кнопку перезагрузки
- Минимальная нагрузка на систему
- Не требует настройки
- Не требует закрытия Яндекс Музыки

## Установка

1. Скопируй папку `Fast Reload` в `%AppData%/Roaming/PulseSync/addons/`
2. Перезапусти PulseSync
3. Аддон автоматически включится

## Использование

Просто нажми **F5** на клавиатуре в любой момент или на кнопку перезагрузки
